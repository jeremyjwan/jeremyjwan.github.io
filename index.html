CS 170: Efficient Algorithms and Intractable Problems

NP-Hard Problem Solver

Python

An NP-hard problem normally takes very long to find a solution for. Given the NP-hard problem of processing graphs with hundreds of forward edges and returning a linearized ordering that yields the most forward edges, an approximation algorithm needs to return a solution within a limited time. By combining Greedy Algorithms with Dynamic Programming, our approximation placed us in the top 20% in a competition for the most forward edges.

Distributed Page Rank Analyzer

Python | Apache Hadoop MapReduce | Amazon EC2 cloud computing

Given a graph showing the connections between web pages, page rank analysis can give the likelihood of a page being visited next based on where we currently are. By using MapReduce and AWS parallel processing, the algorithm can efficiently return the page rank order of a Facebook graph with over 100,000 edges.
